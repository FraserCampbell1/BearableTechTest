## Fraser's Notes

After completing tasks 1 & 2, here are some improvements I would focus on with more time:

- **Typography consistency**  
  Add font sizing to the `Text.tsx` atom instead of using "magic strings" for font sizes. This ensures consistency across the app.

- **Visual design improvements**  
  Introduce more colour to the Mood Entry input component — it currently feels too grey.

- **User experience enhancements**  
  Improve the experience of adding a mood entry by incorporating:
  - Haptic feedback  
  - Sound effects  
  - Animations (e.g. Lottie animations)
